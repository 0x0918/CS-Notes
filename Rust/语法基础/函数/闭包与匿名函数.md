# 匿名函数

Rust 使用闭包 (closure) 来创建匿名函数：

```rust
let num = 5;
let plus_num = |x: i32| x + num;
```

其中闭包`plus_num`借用了它作用域中的`let`绑定`num`。如果要让闭包获得所有权， 可以使用`move`关键字：

```rust
let mut num = 5;

{
    let mut add_num = move |x: i32| num += x;   // 闭包通过move获取了num的所有权

    add_num(5);
}

// 下面的num在被move之后还能继续使用是因为其实现了Copy特性
assert_eq!(5, num);
```
